# NLP-based-Content-Recommendation-System-using-TF-IDF-and-cosine-similarity
It is a foundational NLP project focused on text vectorization and similarity, which is widely used in real-world recommender system.

## Problem Statement
Users struggle to find movies aligned with their interests.

## Solution
Built a content-based recommendation system using NLP techniques.

## Dataset
MovieLens ml-latest-small dataset.

## Technologies Used
Python, Pandas, Scikit-learn, TF-IDF, Cosine Similarity

## Features
- Genre-based recommendations
- Rating analysis
- Data visualization
